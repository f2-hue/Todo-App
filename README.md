# Tasks — A Todo App
------------------------------------------------------------------
A clean, minimal todo app with localStorage persistence. Built with vanilla 

## Extension
- `index.html` — semantic HTML structure
- `styles.css` — CSS custom properties, no framework
- `app.js` — entry point, event wiring
- `tasks.js` — pure state store (no DOM, no globals)
- `ui.js` — DOM rendering helpers
- `storage.js` — localStorage read/write

## Run 
No build step needed — just open `index.html` in a integrated WebBrowser, or Live server # or python3 -m http.server 8080

## Deploy to GitHub Pages
1. Push to `main` branch on GitHub
2. Host online by using Github Pages
